# PyMD5

## What can I do?

Generate MD5 for string or file.

## Installation:

Clone me to anywhere:

`git clone https://github.com/ITXiaoPang/PyMD5.git`

------

Copy me to the /usr/local/bin

``` shell
cd PyMD5
cp -a pymd5.py /usr/local/bin
chmod +x /usr/local/bin/pymd5.py
```

## How to use?

``` shell
pymd5.py string
pymd5.py file
```

## Enjoy it.
